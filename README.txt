# Kaizen Quests (MAUI, SQLite)
Kurztool zum Organisieren von Aufgaben mit verschachtelten Listen.

**Stack:** C#/.NET MAUI, SQLite
**Features:** Drag&Drop, CRUD, persistente Speicherung
**Start:** Visual Studio öffnen → Build → Run

Architektur (grob):
UI → ViewModel (MVVM) → Repository → SQLite

Lessons learned:
- Drag&Drop in verschachtelten Strukturen
- Persistenz & Datenmodellierung für einfache CRUD-Fälle
